# ToDoApp

### IOS Bootcamp Programı | Swift | 2024

Bu proje, **Kasım Adal** tarafından verilen IOS Bootcamp Programı kapsamında geliştirilmiştir. **ToDoApp**, kullanıcıların günlük görevlerini yönetmelerine olanak tanıyan basit bir görev takip uygulamasıdır.

## Projede Kullanılan Yapılar ve Teknolojiler

- **MVVC Mimari Deseni:** Uygulamanın kod yapısını daha düzenli ve yönetilebilir kılmak için kullanılmıştır.
- **RxSwift:** Reactive programming paradigması ile veri akışlarını ve olayları yönetmek için kullanılmıştır.
- **DbBrowser SQLite:** Uygulamanın veritabanı yönetimi için SQLite kullanılmıştır.

## Özellikler

- Görev Ekleme: Kullanıcılar yeni görevler ekleyebilir.
- Görev Güncelleme: Eklenen görevler düzenlenebilir.
- Görev Silme: Kullanıcılar tamamlanan görevleri silebilir.
- Görev Listeleme: Tüm görevler, düzenli bir şekilde listelenir.
